Gauss Elimination (row reduction)

Gauss elimination is an algorithm for solving systems of linear equations. It uses row operations to convert the associated matrix of coeficients into non-reduced row echelon form (upper triangular). Gauss-Jordan elimination uses row operations to convert the matrix is converted into reduced row echelon form.

A matrix is in row echelon form if it satisfies the following conditions:

  • All nonzero rows (rows with at least one nonzero element) are above any rows of all zeroes (all zero rows, if any, belong at the bottom of the matrix).
  • The leading coefficient (the first nonzero number from the left, also called the pivot) of a nonzero row is always strictly to the right of the leading coefficient of the row above it
  • The leading coefficient must be 1.

A matrix is in reduced row echelon form (also called row canonical form) if it satisfies the following conditions:

  • It is in row echelon form.
  • Every leading coefficient is 1 and is the only nonzero entry in its column.

Solve a system of linear equations A*x=b using elimination method steps:

  1. Create the augmented matrix [A | b]
  2. Apply elementary row oprations until augmented matrix is in reduce row echelon form:
    • Switch two rows
    • Multiply a srow by a scalar
    • Add a multiple of a row to a multiple of another row
